# Mobile Tab Navigation
Project #38 in Brad Traversy's 50 Projects in 50 Days. This project is a Mobile looking UI which changes the image when any of the icons are clicked.

![Mobile Tab Navigation](screenshot.jpg)

## Built With
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Install
```
    $ git clone git@github.com:samiiexx/sam-js-38.git
    $ cd main
```
